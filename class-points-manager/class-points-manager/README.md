## 🔄 V2 版本无损升级指南

如果您之前使用的是 Lite 版（旧版），可以按照以下步骤完美迁移：
1. **关闭程序**：确保旧版和新版程序都处于关闭状态。
2. **迁移核心数据**：将旧版文件夹中的 **`data`** 文件夹整个复制，粘贴到新版根目录下覆盖同名文件夹。
3. **迁移奖品图片**：将旧版文件夹中的 **`static/uploads`** 文件夹复制，粘贴到新版的 **`_internal/static/`** 目录下覆盖同名文件夹。
4. **启动新版**：运行 `.exe`，原本的学生数据、分数、小组以及您上传的奖品图片将全部自动同步。

## 📖 详细操作指南 (基础功能)

### 第一步：班级与学生初始化
1.  **进入管理**：点击导航栏的 **“班级管理”** 或 **“学生管理”**。

*   **排行榜**：支持切换 **“本周”、“本月” 或 “总榜”**。

## 📱 手机/平板同步使用 (局域网)



如果您想在教室里拿着手机走动评分，只需确保手机和电脑连接在**同一个 Wi-Fi** 下：

1. **获取 IP**：在电脑命令行输入 `ipconfig` 找到 IPv4 地址（如 `192.168.1.5`）。

2. **手机访问**：在手机浏览器输入 `http://192.168.1.5:5001`。

3. **小技巧**：在手机浏览器中选择 **“添加到主屏幕”**，以后可以像点击 App 一样直接打开。

## 🌐 进阶：内网穿透 (让学生/家长在校外查看)



如果您希望在校外（非局域网）也能访问积分系统，推荐使用 **Ngrok** 进行内网穿透：



1. **注册账号**：前往 [ngrok.com](https://ngrok.com/) 注册并获取您的 **Authtoken**。

2. **下载客户端**：下载并解压 ngrok 客户端。

3. **配置 Token**：在 ngrok 所在文件夹打开命令行，运行：

   `.\ngrok config add-authtoken 您的Token`

4. **启动映射**：在系统运行的情况下，执行：

   `.\ngrok http 5001`

5. **分享链接**：将 ngrok 界面显示的 `https://xxxx.ngrok-free.app` 链接发送给需要的用户即可。

   *注：免费版每次重启 ngrok 链接会变，建议保持映射窗口不关闭。*
