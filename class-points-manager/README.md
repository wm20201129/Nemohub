# 班级积分管理系统 V3 (单班级专业版)

这是一个专为教师设计的轻量级班级管理与激励工具。系统采用 Python Flask 框架开发，内置 SQLite 数据库，旨在通过积分激励、奖品兑换、互动拍卖和悬赏任务等多种形式，提升班级管理效率与学生参与度。

## 🚀 核心功能

### 1. 班级与学生管理
- **灵活导入**：支持通过 Excel 模板批量导入学生名单（姓名、学号、分组、初始积分）。
- **动态分组**：支持小组化管理，自动计算小组平均分并生成小组排行榜。
- **信息维护**：实时编辑学生及小组信息，支持快速增减员。

### 2. 积分与审核体系
- **科学评分库**：内置包含学业、纪律、活动、德育在内的多维度评分标准，支持自定义修改。
- **基本准则模式**：特有“达标奖励”逻辑，对违纪者扣分的同时，自动为其余达标同学发放奖励分。
- **多级审核**：学生申报加分需经教师在后台审核，确保数据公正权威。
- **快捷加分**：集成“随机点名”功能，支持幸运抽中后的快速一键加分。

### 3. 激励与互动模块
- **荣誉榜单**：实时展示个人积分榜、小组积分榜，区分“荣誉榜（正向）”与“违纪榜（反向）”。
- **奖品小铺**：教师可发布奖品（支持图片），学生通过累积积分进行兑换，支持库存管理。
- **互动拍卖**：支持限时拍卖特殊奖品，学生实时竞价。
- **悬赏任务**：发布个人或小组悬赏任务，设定目标分值与截止日期，自动跟踪进度。

### 4. 系统特性
- **移动端适配**：内置自动获取局域网 IP 功能，支持手机扫码或输入 IP 访问，方便教师在教室内移动操作。
- **内网穿透**：集成 Ngrok 插件，一键开启外网访问。
- **数据安全**：关键操作设有登录密码，支持一键备份导出积分数据及标准库。

## 🛠️ 技术栈
- **后端**: Python (Flask)
- **数据库**: SQLite3
- **前端**: HTML5, CSS3, JavaScript (配合多套管理端与学生端模板)
- **工具**: Openpyxl (Excel 处理), Ngrok (穿透服务)

## 📁 目录结构说明
- `app.py`: 系统主程序，包含所有 API 路由与业务逻辑。
- `config.py`: 环境配置与路径管理。
- `data/`: 存放数据库文件 `class_points.db` 及系统配置文件。
- `static/`: 存放 CSS、JS 静态资源及用户上传的奖品图片。
- `templates/`: 存放所有功能页面的 HTML 模板。
- `password.txt`: 存放系统登录密码（默认为 `123456`）。

## 📝 使用说明
1. 运行 `app.py` 或启动生成的 `.exe` 文件。
2. 首次运行会提示进行基础设置（班级名称、教师姓名）。
3. 默认登录密码见 `password.txt`。
4. 建议定期通过后台导出功能备份学生积分数据。
